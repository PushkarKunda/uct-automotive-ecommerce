# Automotive Parts E-commerce Project

## Overview
This is a comprehensive full-stack e-commerce web application tailored for automotive buyers. Built with the MERN stack (MongoDB, Express, React, Node.js), this platform provides an intuitive shopping experience specifically designed to handle intelligent vehicle fitment logic to help customers find the right parts (by Make, Model, and Year).

## Features
- **Intelligent Search & Vehicle Fitment**: Help users find compatible parts efficiently.
- **Product Catalog**: Detailed part specifications, images, and categorizations (e.g., brake pads, batteries).
- **Shopping Cart**: Persistent cart state across user sessions.
- **User Accounts**: Registration, authentication, and order history tracking.
- **Admin Dashboard**: Manage inventory, view sales, and process orders.
- **Secure Payments**: Integrated Stripe checkouts.
- **Automated Emails**: Order confirmations and tracking via NodeMailer.

## Tech Stack
- **Frontend**: React.js (Vite), TailwindCSS, Redux Toolkit, React Router
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Authentication**: JSON Web Tokens (JWT)
- **Payment Processing**: Stripe
